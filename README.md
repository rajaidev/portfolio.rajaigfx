# Rajai GFX — Graphic Design Portfolio

A modern, responsive portfolio website showcasing graphic design work across branding, packaging, print, editorial, and digital experiences.

## 🎨 Features

- **Responsive Design** — Mobile-first approach with breakpoints at 960px and 640px
- **Dark/Light Mode** — Theme toggle with localStorage persistence
- **Custom Cursor** — Smooth custom cursor with ring and dot animations
- **Scroll Animations** — IntersectionObserver-based reveal animations
- **Multi-Page Projects** — Dedicated project case study pages with galleries
- **No Dependencies** — Pure HTML, CSS (with custom properties), and vanilla JavaScript
- **Performance** — Optimized animations using requestAnimationFrame

## 🛠 Technologies

- **HTML5** — Semantic markup with ARIA labels
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **JavaScript (ES5)** — No frameworks, vanilla implementation
- **Google Fonts** — Playfair Display + Outfit

## ⚡ Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process needed — works instantly!

## 🎯 Features Detail

### Theme System
- Light/Dark mode toggle
- Persists to localStorage
- Respects OS color scheme preference

### Custom Cursor
- Ring cursor (36px bordered circle)
- Dot cursor (5px filled circle)
- Smooth lag effect with requestAnimationFrame
- Automatically hides on clickable elements

### Animations
- Scroll-triggered reveals with IntersectionObserver
- Staggered animations for grouped elements
- Hero typing effect with 4 rotating role descriptions
- Project card 3D tilt on mouse movement
- Counter animations for stats

### Navigation
- Active link detection based on scroll position
- Mobile hamburger menu with drawer
- Back-to-top links on multi-page projects
- Consistent navbar/footer across all pages
